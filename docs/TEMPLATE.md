## テンプレート使用上の注意

1. **導入問題は必須**: 日常に根ざした問いから始める
2. **Mermaid図は積極活用**: フレームワークは視覚化すると理解しやすい
3. **具体例は実務に即したもの**: 抽象的な説明だけでなく、実際に使える形で
4. **関連概念でつなげる**: 知識のネットワークを構築
5. **参考文献は任意**: 必要に応じて記載

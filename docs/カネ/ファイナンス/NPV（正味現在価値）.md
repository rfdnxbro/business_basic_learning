# NPV（正味現在価値）

## 導入問題

> 今日の100万円と1年後の100万円、どちらが価値があるでしょうか？
>
> また、ある投資プロジェクトに初期投資1,000万円が必要で、今後3年間で毎年500万円のキャッシュフローが見込めるとします。この投資は実行すべきでしょうか？

## 考えるポイント

- 同じ金額でも「時間」によって価値が変わる理由を考えてみよう
- 今日もらえるお金なら、銀行に預けて利子を得られる
- インフレや投資機会の違いも影響する
- 将来のキャッシュフローを「今の価値」に換算できれば、投資判断ができるはずである

## 解説

### 用語定義

**NPV（Net Present Value：正味現在価値）** とは、投資プロジェクトから将来得られるキャッシュフローを現在価値に割り引いて合計し、初期投資額を差し引いた値です。NPVがプラスであれば投資価値があり、マイナスであれば投資価値がないと判断します。

### 背景・なぜ重要か

お金には「時間的価値」があります。同じ100万円でも、今すぐ手に入れば運用できるため、将来の100万円よりも価値が高いのです。

NPVは、この貨幣の時間価値を考慮した投資判断の基本指標であり、以下のような場面で活用されます。

- 設備投資の可否判断
- M&Aの企業価値評価
- プロジェクト選択の優先順位付け
- 事業ポートフォリオの見直し

## 詳細説明

### NPVの計算式

NPVは次の式で計算されます。

```
NPV = Σ(CFt / (1 + r)^t) - 初期投資額

CFt: t年後のキャッシュフロー
r: 割引率
t: 年数
```

### 割引率の考え方

割引率は、将来のキャッシュフローを現在価値に換算するための利率です。以下の要素を考慮して決定します。

- **資本コスト**: 資金調達にかかるコスト（借入金利、株主が期待する収益率など）
- **リスクプレミアム**: プロジェクトのリスクに応じた上乗せ率
- **機会費用**: その資金を別の投資に使った場合の収益率

一般的に、リスクが高いプロジェクトほど高い割引率を設定します。

### NPVによる投資判断の基準

- **NPV > 0**: 投資価値あり（実行すべき）
- **NPV = 0**: 投資価値なし（実行しても損得なし）
- **NPV < 0**: 投資価値なし（実行すべきでない）

複数のプロジェクトから選択する場合は、NPVが最も大きいものを優先します。

### IRR（内部収益率）との比較

NPVと並んでよく使われる指標に**IRR（Internal Rate of Return：内部収益率）** があります。IRRはNPVがゼロになる割引率のことで、投資の収益率を表します。

| 指標 | 意味 | メリット | デメリット |
|-----|------|---------|----------|
| NPV | 投資による価値増加額 | 金額で評価でき、複数案の比較が容易 | 割引率の設定が難しい |
| IRR | 投資の収益率 | 利回りで直感的に理解しやすい | 複数の解が存在する場合あり |

理論的にはNPVの方が優れた指標とされています。なぜなら、IRRは規模の違うプロジェクトを比較する際に誤った判断を導く可能性があるためです。

## 具体例・ケーススタディ

### 例1：工場の設備投資

ある製造業が、新しい製造ラインに5,000万円の投資を検討しています。この設備は5年間使用でき、毎年1,500万円のキャッシュフローを生み出すと予想されます。割引率を8%として、投資の妥当性を判断しましょう。

**計算**

```
NPV = 1,500/(1.08)^1 + 1,500/(1.08)^2 + 1,500/(1.08)^3
    + 1,500/(1.08)^4 + 1,500/(1.08)^5 - 5,000

    = 1,389 + 1,286 + 1,191 + 1,103 + 1,021 - 5,000
    = 5,990 - 5,000
    = +990万円
```

NPVがプラスなので、この投資は実行すべきと判断できます。

### 例2：複数プロジェクトの比較

企業がA案とB案の2つのプロジェクトを検討しています（割引率10%）。

**A案**
- 初期投資：1,000万円
- 1年後CF：600万円
- 2年後CF：600万円

**B案**
- 初期投資：2,000万円
- 1年後CF：1,100万円
- 2年後CF：1,200万円

**計算**

```
A案NPV = 600/1.1 + 600/1.21 - 1,000
       = 545 + 496 - 1,000
       = +41万円

B案NPV = 1,100/1.1 + 1,200/1.21 - 2,000
       = 1,000 + 992 - 2,000
       = -8万円
```

この場合、B案は投資価値がないため、A案を選択すべきです。

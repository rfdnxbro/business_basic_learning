# ビジネス基礎知識 学習教材

ビジネスパーソン向けのビジネス知識学習教材です。「ヒト」「モノ」「カネ」「思考・志」の4カテゴリで、概念やフレームワークを体系的に学べます。

## 対象者

- ビジネスの基礎を学びたい方
- フレームワークや理論を実務に活かしたい方
- 体系的にビジネス知識を整理したい方

## カテゴリ構成

### ヒト

組織と人に関する知識。

| 中カテゴリ | 概要 |
|-----------|------|
| 組織行動・リーダーシップ | 組織の中で人がどう動くか、リーダーの役割とは何か |
| 人材マネジメント | 採用・育成・評価・報酬など人材に関する仕組み |

### モノ

商品・サービスと市場に関する知識。

| 中カテゴリ | 概要 |
|-----------|------|
| マーケティング | 顧客理解、市場分析、製品戦略、プロモーション |
| 経営戦略 | 競争優位の構築、事業ポートフォリオ、成長戦略 |

### カネ

財務・会計に関する知識。

| 中カテゴリ | 概要 |
|-----------|------|
| アカウンティング | 財務諸表の読み方、管理会計、原価計算 |
| ファイナンス | 資金調達、投資判断、企業価値評価 |

### 思考・志

思考法と倫理に関する知識。

| 中カテゴリ | 概要 |
|-----------|------|
| クリティカルシンキング | 論理的思考、問題解決、意思決定 |
| ビジネス倫理 | 企業の社会的責任、コンプライアンス、ステークホルダー |

## 記事フォーマット

各記事は以下の構成で統一されています。詳細は [TEMPLATE.md](docs/TEMPLATE.md) を参照してください。

1. **導入問題** - 日常の疑問から始める問いかけ
2. **考えるポイント** - 回答を導くヒント
3. **解説・用語定義** - 概念の説明
4. **詳細説明** - 理論やフレームワークの詳細
5. **具体例・ケーススタディ** - 実際の適用例
6. **関連概念** - 他の記事へのリンク
7. **参考文献** - 出典（任意）

フレームワークの解説にはMermaid図を活用しています。
